Назначение

Массовые операции с карточками (черновик)

Правила отбора целей: сначала отмеченные чекбоксами, иначе артикул/серию в поле выше, иначе текущие фильтры, иначе все позиции. Пустое поле + кнопка — сброс своего заголовка.

Для «Пунктов карточки»: пустой список + кнопка — сброс своих пунктов карточки (вернутся стандартные для family). Эти пункты едины для всей позиции и действуют на все фасовки.

Массовые фасовки — в блоке «Фасовки для сайта и каталога» ниже. Массовые операции с фото в этом интерфейсе отключены.

Фасовки для сайта и каталога

Фасовки для сайта и каталога (массово)

Редактирование и удаление строк в таблице сначала только на экране; «Таблицу фасовок в черновик» сохраняет состав в черновик для целей массовых операций; «Отменить правки таблицы» возвращает вид к сохранённому черновику. «Сохранить на сайт» вверху применено к сайту только после вашего подтверждения. Базовый список строится по прайсу: веса розницы, ведро/барабан при заданном массаже в каталоге. После смены прайса можно нажать «Как из прайса», затем снова «Таблицу фасовок в черновик», затем «Сохранить на сайт».

↕ Тип Кг (банка) Подпись Подзаголовок Скрыть

Ручные строки поддерживаются: добавьте тип и массу, далее цена фасовки на сайте рассчитывается автоматически от текущей цены за кг и указанной массы.

Классификация фасовок по кг: 0.1–2.7 — Банка мал; 2.8–6 — Банка бол; 7–30 — Ведро; 32–50 — Барабан.

Фасовки для сайта и каталога (выбранная позиция)

Добавление, изменение порядка и удаление строк — для выбранной позиции. Чипы на карточке товара и в каталоге. «Записать в черновик» — правки в админке; «Опубликовать на сайте» — то же, что кнопка «Сохранить на сайт» вверху (для всех изменений черновика).

Прайс-лист и цены

Каталог хранится в products-catalog.json на сервере. После сохранения сайт подхватывает данные через API (главная, прайс, карточки, корзина). Цены за 1 / 2,8 / 20 / 50 кг считаются автоматически от поля «грн/кг с НДС».

Изменения только в браузере, пока не нажмёте «Сохранить каталог».

Базовый формат: id,family,code,name,jarSmallKg,jarBigKg,bucketKg,drumKg,priceNoNdsPerKg,priceNdsPerKg. Можно загрузить CSV, XLS или XLSX; новые строки без id будут добавлены как новые позиции. Дополнительные столбцы из файла автоматически появятся в таблице как редактируемые. При импорте лишние пробелы, неразрывные пробелы и переносы строк в ячейках убираются, текст приводится к единому виду.
